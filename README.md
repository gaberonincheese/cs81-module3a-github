# cs81-module3a-github

A repository for CS81.



Functions in journal.js and their purpose.



* greet: This function greets a user based on the name given (singular parameter).
* sayHello: This function returns a greeting when called (no parameters).
* add: This function adds two numbers based on passed in numbers (two params).
* subtract: This function subtracts numbers based on passed in numbers (two params).
* determineOddOrEven: This function determines if a passed in number is odd or even using modulo (one param).
